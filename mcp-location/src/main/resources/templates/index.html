About This MCP Server

This server provides meeting location suggestions based on user preferences and can book rooms at the location.

MCP (Model Context Protocol) allows AI models to securely connect to external data sources and tools. This server exposes location and room booking capabilities through standardized tools.

Available Tools

all-locations

Get all available meeting locations.

No Parameters
check-room-availability

Check room availability for a specific location, date, time, duration and number of people.

Complex Parameters
book-room

Usage Instructions

For AI Models

http://localhost:8081/mcp/sse
Tool Usage Examples

Use the all-locations tool to retrieve all available meeting locations.

Parameters: None

Returns: List of locations with ID, name, and description.

Use the check-room-availability tool to verify if a room is available.

Required Parameters:

  • locationId - The location identifier
  • date - The requested date
  • startTime - Meeting start time
  • durationInMinutes - Meeting duration
  • requestedNumberOfPeople - Number of attendees

Use the book-room tool to make a room reservation.

Required Parameters:

  • locationId - The location identifier
  • roomId - The specific room to book
  • date - The booking date
  • startTime - Meeting start time
  • durationInMinutes - Meeting duration
  • reference - Booking reference
  • description - Meeting description
