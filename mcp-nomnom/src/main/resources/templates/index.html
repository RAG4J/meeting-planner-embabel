About This MCP Server

AI-powered food ordering service for meetings. Order delicious food and drinks for your team!

MCP (Model Context Protocol) allows AI models to securely connect to external data sources and tools. This server exposes food ordering capabilities through standardized tools powered by Embabel agents.

Available Tools

Finds the best matching product for a provided product name using AI. Handles fuzzy matching and suggests alternatives when exact matches aren't found.

Parameter:

Usage Instructions

For AI Models

Connect to this Embabel MCP server to access food ordering capabilities. The server provides intelligent product matching and order processing.

Tool Usage Example

Use the findBestMatchingProduct tool to find products even with fuzzy or approximate names.

Parameter:

  • providedName - Product name (e.g., "coffee", "burger", "soda")

Returns: Product object with name, description, price, and category.

Example: Search for "coke" and the AI will intelligently match it to "Soda".

The HandleOrderAgent processes food orders intelligently.

Features:

  • Natural language order processing
  • Automatic product matching
  • Quantity handling
  • Order confirmation
