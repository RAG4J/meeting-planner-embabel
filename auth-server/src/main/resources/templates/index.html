Welcome to the Authorization Server

This OAuth2 Authorization Server provides secure authentication and authorization services for the Meeting Planner application ecosystem. It handles user authentication, client authorization, and token management using industry-standard OAuth2 and OpenID Connect protocols.

User Authentication

Secure login system with support for multiple user roles including regular users, meeting planners, and administrators.

OAuth2 & OpenID Connect

Full OAuth2 Authorization Code flow with PKCE support, consent management, and OpenID Connect for identity verification.

Client Management

Registered OAuth2 clients including the Meeting Planner web application and MCP (Model Context Protocol) server integration.

Meeting Permissions

Granular scope-based permissions for meeting operations including read access, meeting creation, and MCP service invocation.

How to Use This Service

  1. For End Users
    Navigate to the Meeting Planner web application and sign in. You'll be redirected here for authentication and consent.
  2. For Developers
    Implement the OAuth2 Authorization Code flow in your client applications using the standard endpoints.
  3. For Administrators
    Use the admin dashboard to manage users, view active sessions, and configure OAuth2 clients.
